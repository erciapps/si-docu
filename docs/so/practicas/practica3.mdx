# Práctica 3
## PRÁCTICA INDIVIDUAL
## Introducción
En esta práctica aprenderás a realizar **copias y clonaciones de discos virtuales** utilizando una máquina virtual en **VMWare** y el entorno **WinPE** con herramientas de gestión de particiones y backups.  
El objetivo es comprender la diferencia entre **clonación** y **copia de seguridad**, así como el proceso de **restauración** de una imagen de disco.  
Durante el ejercicio trabajarás con distintos dispositivos virtuales, aplicando conceptos básicos de almacenamiento y verificación de integridad de datos.

:::info ACTIVIDADES
1. Crea una **clonación** del dispositivo de origen en el dispositivo de destino. Comprueba que el disco destino tiene el mismo número de particiones que el origen.   

2. Crea un backup del dispositivo de origen en el dispositivo de destino. Observa que crea un directorio con varios archivos.  

3. Restaura el backup realizado en otro dispositivo de destino, y observa que se realiza la restauración de igual forma que si fuera una clonación de dispositivo a dispositivo.
:::
